# 🔨 Whack-a-Mole Game

A fun and fast-paced **Whack-a-Mole Game** built using **HTML, CSS, and JavaScript**. Test your reflexes and hit as many moles as you can before time runs out! ⚡🕹️

---

## ✨ Features

* ⚡ Fast reaction-based gameplay
* ⏱️ 30-second countdown timer
* 📊 Real-time score tracking
* 🏆 Best score (high score) system
* 🎯 Interactive game board
* 🎨 Clean and engaging UI

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/whack-a-mole-game.git
```

2. Navigate to the project folder:

```bash
cd whack-a-mole-game
```

3. Open `index.html` in your browser:

```bash
start Whack-a-mole.html
```

---

## 🎮 How to Play

* ▶️ Click on **Start Game**
* 🐹 Moles will randomly appear from holes
* 🔨 Click on the mole to score points
* ⏱️ You have **30 seconds** to score as much as possible
* 🏆 Try to beat your **best score**
