# ⌨️ Typing Speed Test

An interactive **Typing Speed Test** built using **HTML, CSS, and JavaScript**. Improve your typing skills by measuring your speed, accuracy, and consistency in real-time! ⚡🧠

---

## ✨ Features

* ⏱️ 45-second typing test
* 📊 Real-time WPM (Words Per Minute)
* 🎯 Accuracy tracking
* 🏆 Best WPM record
* 🔄 Reset session option
* 💡 Clean and focused UI

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/typing-speed-test.git
```

2. Navigate to the project folder:

```bash
cd typing-speed-test
```

3. Open `index.html` in your browser:

```bash
start Typing-Speed-Test.html
```

---

## 🎮 How to Play

* ▶️ Click on **Start Test**
* ⌨️ Start typing the given text
* 📊 Track your **WPM** and **Accuracy** in real-time
* ⏱️ Complete the test within **45 seconds**
* 🏆 Try to beat your **best WPM**
