# 📝 Todo List App

A clean and interactive **Todo List Application** built using **HTML, CSS, and JavaScript**. Manage your daily tasks efficiently with filtering, tracking, and deletion features! 🚀

---

## ✨ Features

* ➕ Add new tasks easily
* 🗑️ Delete selected or all tasks
* 📂 Filter tasks (All / Pending / Completed)
* ✅ Mark tasks as completed
* 📊 Task counter (Completed & Total)
* 🎨 Clean and user-friendly UI

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/todo-list-app.git
```

2. Navigate to the project folder:

```bash
cd todo-list-app
```

3. Open `To_Do.html` in your browser:

```bash
start To-Do.html
```

## 🎮 How to Use

* ➕ Add a task using the input field
* ✅ Mark tasks as completed
* 🗑️ Delete selected or all tasks
* 🔍 Use filters to view tasks (All / Pending / Completed)
* 📊 Track completed and total tasks
