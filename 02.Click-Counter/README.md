# 🖱️ Click Counter Game

A fun and addictive **Click Counter Game** built using **HTML, CSS, and JavaScript**. Test your speed and see how many clicks you can make before time runs out! ⏱️🔥

---

## ✨ Features

* ⚡ Fast-paced clicking game
* ⏳ 10-second countdown timer
* 📊 Real-time score tracking
* 🏆 High score system
* 🔄 Reset high score option
* 🎯 Simple and responsive UI

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript 

---

```
📦 Click-Counter-Game
 ┣ 📜 Click_Counter.html

 ┗ 📜 Click_Counter.js
```

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/click-counter-game.git
```

2. Navigate to the project folder:

```bash
cd click-counter-game
```

3. Open `index.html` in your browser:

```bash
start Click_Counter.html
```

---

## 🎮 How to Play

* ▶️ Click on **Start Game**
* 🖱️ Click the **Click Me!** button as fast as possible
* ⏱️ You have only **10 seconds**
* 🏆 Beat your **high score**
