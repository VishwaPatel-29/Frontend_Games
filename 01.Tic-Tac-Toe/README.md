# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**. This project is perfect for beginners who want to understand DOM manipulation and basic game logic.

---

## ✨ Features

* 🧠 Two-player gameplay (X vs O)
* 🎯 Winner detection logic
* 🔄 Real-time UI updates
* 💡 Simple and clean design
* ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Navigate to the project folder:

```bash
cd tic-tac-toe
```

3. Open `index.html` in your browser:

```bash
start index.html
```

---

## 🎮 How to Play

* 👥 Two players take turns
* ❌ Player 1: X
* ⭕ Player 2: O
* 🏆 First player to align 3 marks (row, column, or diagonal) wins
