# 🃏 Memory Flip Card Game

A fun and engaging **Memory Flip Card Game** built using **HTML, CSS, and JavaScript**. Test your memory skills by matching pairs of cards within the time limit! 🧠⚡

---

## ✨ Features

* 🧩 Flip and match card pairs
* ⏱️ 60-second countdown timer
* 🔢 Move counter tracking
* 🎯 Pair tracking system
* 🏆 Best score record
* 🔄 Restart & hard reset options
* 🎨 Smooth and interactive UI

---

## 🛠️ Tech Stack

* 🌐 HTML5
* 🎨 CSS3
* ⚙️ JavaScript

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/your-username/memory-flip-card-game.git
```

2. Navigate to the project folder:

```bash
cd memory-flip-card-game
```

3. Open `Memory_FlipCard.html` in your browser:

```bash
start Memory_FlipCard.html
```

---

## 🎮 How to Play

* ▶️ Click **Start** to begin
* 🃏 Flip two cards at a time
* 🎯 Match identical pairs
* ⏱️ Complete the game within **60 seconds**
* 🏆 Try to finish with fewer moves and beat your **best score**
