# 🎮 Frontend Games Collection

A collection of 7 interactive frontend games built using HTML, CSS, and JavaScript.  
This repository contains beginner-to-intermediate level projects focused on DOM manipulation, game logic, and UI design.

---

## 🚀 Games Included

### 1️⃣ Tic-Tac-Toe (Player Name Version)
- Two-player game
- Player name input
- Winner detection logic
- Interactive UI

### 2️⃣ Click Counter Game
- Simple click counter
- Score tracking
- Basic event handling

### 3️⃣ Color Guessing Game
- Random color generation
- RGB value guessing
- Difficulty logic

### 4️⃣ Whack-a-Mole Game
- Timer-based gameplay
- Random mole appearance
- Score tracking system

### 5️⃣ Typing Speed Test
- Real-time typing analysis
- Words per minute (WPM) calculation
- Accuracy tracking

### 6️⃣ Memory Flip Card Game
- Card matching logic
- Flip animation effects
- Win condition detection

### 7️⃣ To-Do Game
- Add tasks
- Delete tasks
- Mark tasks as completed
- Dynamic DOM updates

---

## 🛠️ Technologies Used

- HTML5

- DOM Manipulation
- Event Handling
